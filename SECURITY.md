# Security & Privacy Policy



## Privacy Policy

**We do not collect your data.**



* **Local Processing:** This application runs entirely in your web browser (or on your desktop).

* **No Uploads:** When you "upload" an image, it is processed locally in your device's memory. It is **never** sent to an external server or cloud storage.

* **No Tracking:** We do not use cookies or analytics to track your behavior.



## Security Policy



### Supported Versions



| Version | Supported          |

| ------- | ------------------ |

| 1.0.x   | :white_check_mark: |

| < 1.0   | :x:                |



### Reporting a Vulnerability



Since this is a client-side application, most server-side vulnerabilities do not apply. However, if you find a security issue in the logic:



1.  Please do **not** open a public issue.

2.  Email the maintainer at `contact@yigitguven.net`.

3.  We will review it and patch it promptly.
